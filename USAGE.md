## 1. 快速开始

### 1.1 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / Windows 11 |
| AutoCAD | 2014 或更高版本（32位/64位均可） |
| Python | 不需要（使用打包版 EXE） |

### 1.2 启动方式

**方式一（推荐）：直接双击 EXE**

```
releases\v1.0.0\仪表自动编号_v1.0.0.exe
```

**方式二：Python 源码运行**

```bash
python run.py
```

### 1.3 典型操作流程（5 步）

```
① 连接 AutoCAD + 框选块
    ↓
② 选择属性标签（仪表类型、编号）
    ↓
③ 勾选需要编号的仪表类型
    ↓
④ 配置编号格式
    ↓
⑤ 执行编号
```

## 2. 详细步骤说明

### 步骤 ① — 连接 AutoCAD 并框选块

打开插件后，第一个页面如下：

```
┌──────────────────────────────────────┐
│ ① 连接 AutoCAD 并框选块              │
│                                      │
│ 在 AutoCAD 中选中带属性的仪表块，     │
│ 然后点击下方按钮读取。               │
│                                      │
│  🔴 未连接                           │
│                                      │
│  [🔌 连接 AutoCAD]                   │
│  [📦 框选块]  [📋 读取当前选择]      │
└──────────────────────────────────────┘
```

#### 操作方式一：框选块

1. 点击「🔌 连接 AutoCAD」→ 插件自动连接正在运行的 AutoCAD
2. 点击「📦 框选块」→ 切换到 AutoCAD
3. 在 AutoCAD 中框选需要编号的仪表块，右键结束选择
4. 插件自动过滤出**带属性的块**，显示 `✅ 已选中 N 个带属性块`

#### 操作方式二：读取当前选择（推荐）

1. 点击「🔌 连接 AutoCAD」
2. **先在 AutoCAD 中选中**需要编号的块（按住 Ctrl 多选，或框选）
3. 回到插件，点击「📋 读取当前选择」
4. 插件读取当前蓝色夹点选中的对象，过滤出带属性块

> **提示：** 两种方式效果相同。框选方式更直接，读取方式更灵活。

#### 按钮状态说明

| 状态 | 连接按钮 | 框选按钮 | 读取按钮 |
|------|----------|----------|----------|
| 未连接 | 可用 | 灰色 | 灰色 |
| 已连接 | 灰色 | 可用 | 可用 |
| 已选中 | 灰色 | 灰色（再次框选会先清除旧数据） | 可用 |

### 步骤 ② — 选择属性标签

```
┌──────────────────────────────────────┐
│ ② 选择属性标签                       │
│                                      │
│ ① 仪表类型标签：[SIGNAL  ▼]         │
│    示例值: PT, TE, TT, PG...         │
│                                      │
│ ② 编号标签：[TAG        ▼]          │
│    示例值: 001, 002, 003...          │
│                                      │
│ ⏩ 将用「SIGNAL」识别类型，           │
│    写入编号到「TAG」                  │
└──────────────────────────────────────┘
```

- **仪表类型标签**：选择哪个属性标签的值代表仪表类型（如 `SIGNAL`、`TYPE`）
- **编号标签**：选择哪个属性标签用于存放编号结果（如 `TAG`、`NUMBER`）
- 两个标签**不能相同**
- 下拉框下方显示示例值，帮助判断

## 3. 编号配置详解

### 编号前缀

每个编号前面的英文字母，用于区分仪表类型。

| 示例 | 结果 |
|------|------|
| 留空 | `001`, `002`… |
| `PT-` | `PT-001`, `PT-002`… |
| `TE-` | `TE-001`, `TE-002`… |
| `PISA-` | `PISA-001`, `PISA-002`… |

### 起始编号

从哪个数字开始编号。如设为 `100`，则编号为 `PT-100`、`PT-101`…

### 补零位数

编号中数字部分的固定位数，不足补零。

| 位数 | 示例 |
|------|------|
| 0（不补） | `1`, `2`, … `11`, `12` |
| 2 | `01`, `02`, … `11`, `12` |
| 3 | `001`, `002`, … `011`, `012` |

### Y 容差（单位：毫米）

决定哪些块算"同一行"。Y 坐标差在此范围内视为同行，按 X 从左到右排序；超过则进入下一行。

- 图纸单位是毫米时，默认 `50mm` 通常够用
- 如果同行块 Y 坐标偏差大，适当增大容差
- 如果不同行的块 Y 太接近，适当减小容差

### 自动顺延

勾选后，插件会扫描图中已有编号的最大值，从 `最大值+1` 开始编号。此时「起始编号」字段被忽略。

> **示例：** 已有编号到 `PT-012`，自动顺延后从 `PT-013` 开始

## 4. 常见问题

### Q1: 点击「连接 AutoCAD」提示"无法连接到 AutoCAD"

**原因：** AutoCAD 未运行，或 COM 注册异常。

**解决：**
1. 确保 AutoCAD 已打开
2. 确保不是"以管理员身份运行" AutoCAD 而插件不是（或反过来）
3. 重启 AutoCAD 再试

### Q2: 编号写入后 AutoCAD 中没有变化

**原因：** AutoCAD 显示未刷新。

**解决：** 插件会自动执行 `REGEN` 命令。如仍有问题，在 CAD 命令行输入 `REGEN` 回车。

### Q3: 框选后提示"未选中任何带属性的块"

**原因：** 选中的块没有属性标签（Attribute），或者选中的不是块。

**解决：** 确保框选的是**带属性的块参照**（INSERT 实体），而不是普通图形。

### Q4: 点击"框选块"后 AutoCAD 没有反应

**原因：** AutoCAD 可能被其他对话框阻塞。

**解决：** 关闭 AutoCAD 中其他打开的对话框（属性面板、图层管理器等），然后重试。

### Q5: 编号顺序不对

**原因：** Y 容差设置不合适。

**解决：** 增大 Y 容差可以让更多块视为同一行。查看图纸中同行块的 Y 坐标差异，设置合适的容差值。

### Q6: 想重新开始，怎么做？

点击「← 上一步」回到步骤① → 点击「📦 框选块」或「📋 读取当前选择」→ 插件自动清除旧数据，进入全新流程。

### Q7: EXE 打不开？

**解决：** 尝试右键 EXE → "以管理员身份运行"。或使用源码方式 `python run.py`。
